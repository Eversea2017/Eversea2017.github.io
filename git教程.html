安装Git

我是在win7下安装的，在这里的Downloads选择Windows，然后就会下载了，下载好之后的是个exe程序，双击它然后按步骤提示一步步安装就可以了，安装成功之后Win+R调出cmd界面，输入git –version命令，如下，说明git已经正确安装在你的电脑里了，你可以正常使用了。

GitHub Pages

我们写好一个网页，要想别人在网上也能看到的话，我们需要买一个服务器，搭建好web服务，上传你写好的页面代码，但GitHub有一个强大的功能，它能直接展现出你写好的网页代码效果，即GitHub Pages。
先定位到你的仓库主页，然后点击最后边的setting，然后往下拉有一个GitHub Pages，

点击none选取里面的master branch，下面还可以选择你GitHub Pages页面的主题，点击save，然后刷新一下再拉到GitHub Pages这里会看到这么一句话

点击这个链接，你就可以看到你上传代码的效果了，当然要想实现这个GitHub Pages ，必须要在项目首页放一个index
